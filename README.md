# Tic-Tac-Toe 

#### How to run?
To challenge yourself playing against AI, you can use the bash-script below: 
```
./play.sh
```
Also you can do it manually: 
```
go build -o bin/game.out src/game.go && ./bin/game.out
```
Or whatever you want...
